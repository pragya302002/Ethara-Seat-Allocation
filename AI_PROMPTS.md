## Notes on how this log was produced

Every "bug found and fixed" entry above reflects something that actually
failed when run — reproduced with a real command or a real deployment log,
not inferred or assumed. Entries 1–20 were caught and fixed in the
assistant's own sandbox during original development; entries 21–37 were
caught live during the user's actual Windows local setup and Railway/Vercel
deployment, diagnosed from real terminal output, Railway deploy logs, and
browser behavior at each step. Where a limitation couldn't be resolved
in-session at the time (the AI Assistant's live round-trip, since no working
LLM API key was available in the original build sandbox), that was stated
plainly rather than claimed as tested — and was fully resolved and
confirmed working end-to-end during the deployment session (entries 20, 29,
33, 35). See `docs/DEBUGGING.md` for the full technical detail behind each
bug in this table.
